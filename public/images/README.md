# Images Directory

This directory should contain images from https://labaguettemagiquesarttilman.be/

## Recommended Images to Add:

1. **Hero/Banner Image**: Main restaurant/food image for the homepage
2. **Menu Item Images**: Photos of sandwiches, frites, pitas, etc.
3. **Restaurant Interior/Exterior**: Photos of the actual location
4. **Logo**: If available from the website

## Image Sources:

- Download images directly from https://labaguettemagiquesarttilman.be/
- Right-click on images and "Save image as..." to download
- Organize by category (hero, menu, location, etc.)

## Usage in Code:

Images placed here can be referenced as:
- `/images/filename.jpg` in Next.js
- Or using Next.js Image component: `<Image src="/images/filename.jpg" ... />`
